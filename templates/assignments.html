ENGI 1006 Assignments

  1. Homework 1: This assignment was broken up into a theory section and a coding section. For the theory, we had to answer questions related to binary numbers and algorithms. For the coding section, we wrote code to manipulate strings and lists.

  2. Homework 2: For the theory section of this assignment, we answered questions about sorting algorithms and their effiecieny. For the coding section, we wrote a simulator that predicted the path of a projectile fired straight up, and began to write code for a machine learning library.

  3. Homework 3: For the theory section of the assignment, we answered questions about sets and recursion. For the coding section, we continued working on the machine learning library and wrote a module that would test if a given system percolated or not.

    4. Note: The remainder of the assignments only had coding sections.

  4. Homework 4: For this assignment, we wrote a library that would simulate ENGI 1006 using object oriented programming.

  5. Midterm: In this assignment, we wrote a module that would perform certain operations on images, such as removing objects, making them grayscale, and flipping horizontally.

  6. Homework 5: This assignment saw the completion of the machine learning library. Three machine learning algorithms were implemented, several interesting plots were created, and some advanced statistics were computed using the Pandas, Seaborn, and Scikit-learn libraries.

  7. Final: For the final, we created a web server using Flask. That's what's displaying this page!
